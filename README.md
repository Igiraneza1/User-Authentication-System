# User Authentication System

This is a basic authentication system using Next.js App Router, Clerk for auth.

# Tech Stack

- Next.js (App Router)
- Clerk (Authentication)
- Firebase (Database)
- Tailwind CSS (UI)

# Features

 User Registration & Login with Clerk
 Protected Dashboard Page
 Logout
 Tailwind UI
